Featured Project

Project ML Racing

A personal project where I built a Unity-based racing environment and connected it to a PPO reinforcement learning agent that learns to drive using computer vision and vehicle telemetry. What began as a simple experiment became one of the most engaging learning experiences I've had.

The agent can currently make its way around the track — though, for now, my keyboard driving still beats it. There's plenty left to do before it can truly race competitively, and that's exactly what makes it fun.

What I explored

  • Computer vision with Canny edge detection for lane detection
  • CNN-based lane detection using a MobileNet model
  • Data collection & classification to train the model
  • Unity-to-Python integration for camera & telemetry
  • PPO reinforcement learning & reward shaping
  • Model fine-tuning and debugging the full pipeline
The hardest moments came when the model behaved unexpectedly and I had to properly understand the integration between the CNN, the PPO agent, the observations, and the reward structure.
